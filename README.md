# Customer Churn Analysis

An end-to-end Customer Churn Analysis project built using Python, SQL, Pandas, Matplotlib and Seaborn.

## Project Overview
This project analyzes customer subscription behavior to identify churn patterns, customer risk segments, revenue loss, and business insights.

## Tools Used
- Python
- SQL (SQLite)
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Customer Table
- Subscription Table
- Support Table

## Key Analysis
- Data Cleaning
- SQL Table Integration
- Feature Engineering
- Churn Analysis
- Revenue Analysis
- Customer Segmentation
- Business Insights
- Data Visualization
